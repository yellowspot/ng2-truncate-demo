GitHub Repo

Character truncation

Truncate to 5 characters

<span>{{ "123456789" | truncate : 5 }}</span>

Truncate to 5 characters trailing char counts (options as hash)

<span>{{ "123456789" | truncate : 5: { countTrailing: '…', includeTrailing: true } }}</span>

Don't truncate when text is short than limit

<span>{{ "123456789" | truncate : 20 }}</span>

Change ellipsis characters

<span>{{ "123456789" | truncate : 20 : '.' }}</span>

Words truncation

Truncate to 3 words

<span>{{ "I need to truncate on word number 3" | words : 3 }}</span>

Change ellipsis characters

<span>{{ "I need to truncate on word number 3" | words : 3 : '.' }}</span>
